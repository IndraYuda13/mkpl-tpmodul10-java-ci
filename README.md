# TPModul10 MKPL Java Maven

Project Java Maven sederhana untuk tugas MKPL.

Fitur utama:

- Model serangga, semut, dan lebah.
- Build dengan Maven.
- Testing logic sederhana memakai Java assertion.
- GitHub Actions untuk CI.
- CodeQL Security Scan sebagai tool security selain SonarQube/SonarCloud.

## Run Test

```bash
mvn clean test
```

## Run App

```bash
mvn exec:java
```

## Git Activity Check

```bash
bash scripts/check-git-activity.sh
```
