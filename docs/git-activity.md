# Git Activity

Aktivitas Git yang dilakukan pada project ini:

1. Membuat repository lokal dengan git init.
2. Checkout branch main.
3. Commit import project awal.
4. Push branch main ke GitHub.
5. Membuat branch activity/git-workflow.
6. Commit dokumentasi aktivitas Git.
7. Push branch activity/git-workflow ke GitHub.
8. Merge branch activity/git-workflow ke main.
9. Push hasil merge ke GitHub.
